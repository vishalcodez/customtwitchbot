CUSTOM TWITCH BOT

Usage Guide:


Registration:
Start by using the /register command to link your Twitch account to the bot. Only linked accounts will be registered.
The bot sends a message in DM for registration when anyone joins the server.


Profile Update:
Use /update if you need to modify your registered Twitch account. Follow prompts to ensure accurate information.


Profile Check:
Want to view your details? Use /profile to see your linked Twitch account and current coin balance.


Bot Latency:
Curious about bot responsiveness? Type /ping to get an idea of the bot's latency.


Uptime Check:
Interested in how long the bot has been operational? Use /uptime to see the bot's current uptime.


Settings Access:
Access the bot's settings using /settings. Depending on availability, you can modify settings as needed.


Coin Balance:
To check your earned coins, type /coins. The bot will display your total coins accumulated.


Get Help:
Need assistance? Use /help to receive a list of available commands and their functions.
The bot sends a DM message prompting users to register when they join the server.
